# 合併當前分支到 main

將當前分支合併到 main 分支，並推送到遠端。

## 執行步驟

1. 執行 `git status` 確認當前分支名稱，以及是否有未提交的變更。
   - 如果有未提交的變更，提醒使用者先執行 `/commit-all` 提交後再合併，然後**中止**。
   - 如果當前已在 main 分支，告知使用者「已經在 main 分支上，無需合併」並結束。

2. 記錄當前分支名稱（例如 `dev-v1`）。

3. 切換到 main 分支並拉取最新：

   ```
   git checkout main
   git pull origin main
   ```

4. 合併來源分支：

   ```
   git merge <來源分支名稱>
   ```

5. 如果合併發生衝突：
   - 列出衝突檔案
   - 提醒使用者需要手動解決衝突
   - **中止**後續步驟

6. 合併成功後，推送到遠端：

   ```
   git push origin main
   ```

7. 切換回原本的分支：

   ```
   git checkout <來源分支名稱>
   ```

8. 顯示合併結果摘要：
   - 來源分支 → main
   - 合併的 commit 數量
   - 是否已推送到遠端
